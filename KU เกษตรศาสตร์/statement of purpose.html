สวัสดีครับผมชื่อ นายธนกร ประดับเวช หรือชื่อเล่นว่า กร ผมนั้นเป็นคนที่ชอบลองสิ่งใหม่ๆอยู่เสมอ ชอบที่จะอ่านหนังสือและศึกษาค้นคว้าเกี่ยวกับสิ่งที่น่าสนใจตั้งแต่เด็ก ซึ่งการเขียนโปรแกรมในตอนนั้นเป็นสิ่งที่ใหม่และเป็นสิ่งที่น่าสนใจสำหรับผมเป็นอย่างมาก ความสนใจของผมในด้านการเขียนโปรแกรมได้เพิ่มมากขึ้นเมื่อผมได้เรียนรู้เกี่ยวกับภาษา HTML และ CSS ซึ่งทำให้ผมสามารถสร้างและออกแบบเว็บไซต์ได้เป็นครั้งแรก การเขียนโปรแกรมในภาษาเหล่านี้ไม่เพียงแต่เปิดโอกาสให้ผมได้เรียนรู้การจัดวางโครงสร้างและดีไซน์เท่านั้น แต่ยังทำให้ผมเข้าใจถึงการทำงานของเว็บเบื้องต้น สิ่งนี้จุดประกายความสนใจที่ทำให้ผมเริ่มสำรวจและศึกษาเพิ่มเติมในภาษาโปรแกรมอื่น ๆ รวมถึงเทคโนโลยีที่อยู่เบื้องหลังการทำงานของคอมพิวเตอร์อย่างลึกซึ้งมากยิ่งขึ้น

ผมจึงมีความตั้งใจที่จะศึกษาต่อในสาขาวิศวกรรมคอมพิวเตอร์ คณะวิศวกรรมศาสตร์ มหาวิทยาลัยเกษตรศาสตร์ เพื่อสร้างพื้นฐานและพัฒนาความรู้ทางเทคโนโลยีที่แข็งแกร่ง และต่อยอดการเรียนรู้ที่ผมนั้นเริ่มต้นจากการเขียนภาษา HTML และ CSS ในการออกแบบเว็บไซต์เบื้องต้น ซึ่งประสบการณ์นี้ทำให้ผมได้ตระหนักถึงความสำคัญของโครงสร้างระบบและความสร้างสรรค์ในการพัฒนาแอปพลิเคชันและเทคโนโลยีดิจิทัลอย่างมีประสิทธิภาพ

หลังจากเริ่มเขียนภาษา HTML และ CSS ผมได้ขยายความสนใจไปสู่การศึกษาเชิงลึกในด้านการพัฒนาซอฟต์แวร์และการออกแบบ ซึ่งทำให้ผมมั่นใจว่าการศึกษาต่อในสาขาวิศวกรรมคอมพิวเตอร์ จะช่วยให้ผมสามารถสร้างสรรค์นวัตกรรมที่สามารถแก้ปัญหาจริงในชีวิตประจำวันได้ นอกจากนี้ การได้มีส่วนร่วมในการทำโครงการจริงที่ต้องการการวางแผนเชิงวิศวกรรม ได้ส่งเสริมทักษะในการคิดวิเคราะห์และการจัดการปัญหาที่ยากขึ้นและซับซ้อนให้แก่ผม

CSS นั้นเป็นเครื่องมือที่ทำให้เว็บไซต์ของผมมีความน่าสนใจมากยิ่งขึ้น ผมจึงเริ่มทดลองโดยการปรับแต่งสีสัน รูปแบบตัวอักษร และการจัดวางองค์ประกอบต่างๆ ทำให้ผมตระหนักรู้ว่า การสร้างเว็บไซต์ไม่ใช่แค่การเขียนโค้ด แต่ยังเป็นการแสดงออกถึงความคิดสร้างสรรค์และเอกลักษณ์ส่วนตัวของผม

เมื่อได้เรียนรู้ภาษาโปรแกรมเหล่านี้ ผมจึงเริ่มนำภาษาโปรแกรม มาประยุกต์ใช้กับโครงการระดับประเทศอย่าง
“โครงการอบรมพลเมืองสร้างสรรค์นวัตกรรมต่อต้านการทุจริตในท้องถิ่น รุ่นที่2” ซึ่งทำให้ผมได้ฝึกฝนทักษะในการวางแผนและการออกแบบระบบเว็ปไซต์ที่ใช้งานได้จริง นอกจากนี้ ผมยังได้เรียนรู้เกี่ยวกับการทำงานเป็นทีมและการสื่อสารกับผู้อื่น ซึ่งเป็นทักษะที่สำคัญสำหรับการทำงานในสายวิศวกรรมคอมพิวเตอร์

มหาวิทยาลัยเกษตรศาสตร์เป็นสถาบันการศึกษาที่ผมเชื่อว่ามีคุณภาพในการเรียนการสอนที่ยอดเยี่ยมในด้านวิศวกรรมคอมพิวเตอร์ และเป็นแหล่งที่สามารถส่งเสริมความคิดสร้างสรรค์ผ่านการลงมือปฏิบัติได้อย่างเต็มที่ หลักสูตรของที่นี่จะช่วยให้ผมได้พัฒนาทักษะการเขียนโค้ดเชิงลึก การจัดการข้อมูล และความรู้เกี่ยวกับโครงสร้างพื้นฐานของระบบคอมพิวเตอร์ในระดับที่ลึกซึ้งยิ่งขึ้น ซึ่งล้วนเป็นทักษะที่จำเป็นต่อการก้าวไปสู่การพัฒนาเทคโนโลยีที่มีคุณภาพ

ในอนาคต ผมมีเป้าหมายที่จะเป็นวิศวกรคอมพิวเตอร์ที่สามารถพัฒนานวัตกรรมที่สร้างสรรค์และช่วยเพิ่มคุณภาพชีวิตให้กับสังคม ผมเชื่อว่าการเรียนรู้ในมหาวิทยาลัยเกษตรศาสตร์จะทำให้ผมมีพื้นฐานที่แข็งแกร่งและความสามารถในการนำความรู้มาใช้ให้เกิดประโยชน์ได้อย่างเต็มที่ และผมพร้อมที่จะมุ่งมั่นพัฒนาตนเองเพื่อสร้างสรรค์สิ่งใหม่ ๆ ที่มีคุณค่าในสายงานวิศวกรรมคอมพิวเตอร์
